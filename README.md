<!-- GETTING STARTED -->
## Getting Started



### Prerequisites


* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/jeremaquai/AdoptAPet.git
   ```
2. Install NPM dependancies packages
   ```sh
   npm install
   ```
3. Start the application locally
   ```sh
   npm start
   ```
